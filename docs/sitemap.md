# Target Sitemap

This sitemap is the proposed structure for the rebuilt Klimacontra website.

## Primary Navigation

- `/` — Home
- `/about/` — About Klimacontra
- `/products/` — Product catalogue
- `/solutions/` — Solutions by application
- `/projects/` — Project references and case studies
- `/certifications/` — Certifications and approvals
- `/downloads/` — Catalogues, datasheets, certificates
- `/contact/` — Contact and request forms

## Notes

The existing WordPress export includes duplicate paths such as `diffusers/` and `diffusers-2/`. These should be consolidated into one final product taxonomy and redirected cleanly.
